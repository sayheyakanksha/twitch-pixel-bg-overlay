# Twitch Pixel Animated Background Overlay

A retro-inspired pixel animated background overlay perfect for Twitch stream "Starting Soon" screens. Features smooth animations, a vignette effect, and customizable pixel elements with a nostalgic 8-bit aesthetic.

## Features

**Animated Pixel Grid** – A dynamically moving pixel canvas layer for visual interest  
**Floating Particles** – Slow-moving stars that add depth to the background  
**Vignette Effect** – Subtle darkened edges to frame your content  
**Pixel Bar Animation** – Colorful animated decorative bars with staggered timing  
**Retro Aesthetic** – Press Start 2P font and pixelated styling for that authentic 8-bit vibe  
**Lightweight** – Minimal dependencies, pure HTML/CSS/JavaScript

## Getting Started

Simply open `index.html` in your browser to see the "Stream Starting Soon" overlay in action.

### Customization

- Modify the color scheme by editing the hex colors in the CSS
- Adjust animation speeds by changing the `animation` duration values
- Customize the canvas resolution and opacity in the `#pixels` styles
- Update the content text in the `.content` div
